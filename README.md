# ai-llm-daily

Daily practice tasks on AI/LLMs — building toward an AI/LLM engineering role.

Two tasks per day, committed here to show consistent, active progress.

## Roadmap

- Days 1-7: Python fundamentals through AI-flavored exercises
- Days 8-14: First contact with LLM APIs
- Days 15-21: Prompt engineering
- Days 22-30: RAG & embeddings
- Days 31-45: Applications, tools & agents
- Days 46+: Portfolio projects & job-search prep

Each day's folder (`day-NN-YYYY-MM-DD/`) contains a README with theory, step-by-step instructions, starter code, and a separate solution.
